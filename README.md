## Install

1. Install the Gatsby CLI
   ```
   npm install -g gatsby-cli
   ```
2. Create a new Gatsby site from your CLI:
   ```
   gatsby new <site-name> https://github.com/soniasusanto/ss-portfolio
   ```
3. Start the site:
   ```
   cd <site-name>
   npm install
   gatsby develop
   ```

## Personalization

Edit `config.js` to put up your details.
